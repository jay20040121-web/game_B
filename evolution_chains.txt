================================================================
     像素怪獸 進化鍊總覽 (Evolution Chains Reference)
           最後更新：2026-03-23
================================================================

【說明】
  - 每條線以字母代碼標示 (A/B/AB/C/F/P1/P2/G1/G2/S)
  - 階段時間：Stage1→2 需30秒，Stage2→3 需60秒，Stage3→4 需120秒
  - 格式：階段 | 寶可夢名稱 (PokeAPI ID)

【進化鏈鎖定規則】（2026-03-23 新增）
  - Stage 0 → Stage 1（百變怪階段）：
      所有線可依條件互通分支（F > P > A/B/C）
  - Stage 1 之後（Stage 1→2、2→3、3→4）：
      怪獸只能在自己所屬的進化鏈內繼續進化，
      不可跨線切換。

  各線鎖定行為：
  ┌──────────┬────────────────────────────────────────────┐
  │  線別     │  Stage 1 後的行為                          │
  ├──────────┼────────────────────────────────────────────┤
  │  A/B/C   │  視為同一條「一般線」，三線之間仍可互通      │
  │  F       │  只能在 F 線內（正規或失敗分支）             │
  │  P1      │  只能在 P1 線內（含 P1_SPECIAL 分支）       │
  │  P2      │  只能在 P2 線內（含 P2_SPECIAL 分支）       │
  │  G1/G2   │  完全封閉，不受任何條件影響（D線原有規則）   │
  └──────────┴────────────────────────────────────────────┘

  ※ D線（G1/G2）、F線、P線均無法在 Stage 1 後切換至一般線(A/B/C)，
    一般線(A/B/C)也無法在 Stage 1 後切換至 D/F/P 線。

================================================================
  ★ F 線：格鬥線 / 特訓進化 (Stage 1 → 4)
================================================================

  觸發條件：每個階段內完成足夠次數的特訓小遊戲勝利
    Stage 1→2：stageTrainWins >= 1
    Stage 2→3：stageTrainWins >= 2
    Stage 3→4：stageTrainWins >= 4
  （格鬥線優先於心情/飢餓線；stageTrainWins 每次進化後歸零）
  （若在 F 線但特訓次數不足：將進入特訓失敗分支，無法繼續正規 F 線）

    - 若觸發：重生為 Stage 1，以鬼斯 或 凱西 各 50% 隨機開始

================================================================
  進化條件優先順序（由高→低）
================================================================

  【Stage 0→1 首次分支（百變怪，所有線可觸發）】
  1. F線（格鬥線，stageTrainWins >= 1）
  2. P線（毒化線，mood=0 且 hunger=0）→ P1/P2 各50%
  3. A線（心情 >= 50）
  4. B線（飢餓 >= 50，且心情 < 50）
  5. C線（心情 < 50 且飢餓 < 50，但皆大於0）

  【Stage 1 之後（進化鏈鎖定，只看目前所在線）】
  1. G1/G2   → 沿原線（D線完全封閉）
  2. F系列   → F / F_FAIL1 / F_FAIL2（依 stageTrainWins 判斷）
  3. P1系列  → P1 / P1_SPECIAL（依 mood > 80 判斷）
  4. P2系列  → P2 / P2_SPECIAL（依 mood > 80 判斷）
  5. A/B/C   → 一般線三線互通（依 m/h 切換）
               └─ Stage 3→4 特殊：mood<50 && hunger<50 → DRAGON
